# Weather Agent

Get current weather and short forecasts quickly. The Weather Agent is built for fast, simple answers when you need to check conditions or plan around the forecast.

## What It Helps With

- Checking the current weather for a city, town, region, or airport
- Checking weather for landmarks and other named places
- Checking weather for airport codes
- Checking the moon phase
- Getting a short forecast for the next few days
- Handling different unit preferences and languages when requested
- Looking up travel weather before a trip
- Answering quick weather questions in chat

## Good For

- Daily planning
- Travel preparation
- Outdoor events
- Quick location-based weather checks

## Why People Use It

- Fast answers without extra setup
- Useful for simple weather questions
- Easy to use for both current conditions and short forecasts

## Notes

- Best for concise weather lookups rather than detailed meteorological analysis
- Moon phase questions do not require a location
- No special setup is required
